# متجر هايست - Hayest Store

منصة التجارة الإلكترونية المتقدمة الخاصة بـ **Highest Store**، المبنية باستخدام إطار العمل Bagisto 2.4.x.

## المميزات الرئيسية للمنصة:
* **تكامل AliExpress**: نظام مزامنة المنتجات وتلبية الطلبات تلقائياً مع AliExpress.
* **إدارة كاملة للمخزون والشحن**: لوحة تحكم متقدمة لإدارة المخزون، الطلبات، الفواتير، والشحنات.
* **دعم متعدد اللغات والعملات**: مهيأ بالكامل للغتين العربية والإنجليزية.
* **أداء فائق وسرعة استجابة**: تصميم حديث ومتجاوب مع جميع الأجهزة.

---

### بيئة التشغيل والتطوير المحلّي:
* **Laravel version**: 11.x
* **PHP version**: >= 8.3
* **eCommerce Core**: Bagisto 2.4.x
