## 🚀 كيفية التشغيل

### 1. تشغيل السيرفر:
```bash
php artisan serve
```
**الرابط:** http://127.0.0.1:8000

### 2. مسح الكاش (إذا لزم الأمر):
```bash
php artisan config:clear
php artisan cache:clear
php artisan view:clear
```

### 3. تشغيل Migrations (إذا لزم الأمر):
```bash
php artisan migrate
```
